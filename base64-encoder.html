What is Base64?

Base64 is a binary-to-text encoding scheme that represents binary data as ASCII text. It's commonly used to embed images in HTML/CSS, encode email attachments (MIME), transmit data in URLs, and store binary data in JSON or XML. Each Base64 character represents 6 bits of data, so encoded output is roughly 33% larger than the input.
